# agronomy-cowpea-eif4e
Repository containing data and structural models for the multi-omics characterization of eIF4E isoforms in cowpea (Vigna unguiculata). Article "Unveiling Three Functionally Diverse Isoforms of eIF4E in Cowpea Through a Multi-Omics Approach" published in Agronomy (MDPI) (DOI: 10.3390/xxxxx).
